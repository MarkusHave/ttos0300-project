# Käyttöliittymä ohjelmointi kurssin loppuprojekti

Projekti on C# WPF sovellus, joka käyttää MySQL tietokantaa.  
Projektin tarkoituksena oli tehdä tuntikirjaus ohjelma, mistä näkee millon kukakin yrityksen työntekijä on tullut töihin,
pitänyt taukoja, lähtenyt kotiin sekä kuinka paljon mahd. ylityötunteja on kertynyt.  
Johtajat näkevat alaistensa työajat ja tunnit ja pystyvät muokkaamaan niitä tarvittaessa. Johtajat eivät näe toistensa tunteja ja työaikoja.  
Normaalit työntekijät eivät näe muuta kuin omat työtunnit.
Ohjelmalla voi myös lisätä, poistaa ja muokata työntekijoitä tietokannasta.
